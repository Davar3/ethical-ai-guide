# Task 11: LaTeX Formatting

Create professional academic documents using LaTeX.

## What is LaTeX?

A professional document preparation system used for academic papers, especially those with equations, tables, and figures.

## Why Use LaTeX

- Professional typesetting
- Automatic formatting
- Better equations
- Required by many journals

## Quick LaTeX Tips

### Basic Structure

```latex
\documentclass{article}
\usepackage{booktabs}

\title{Your Title}
\author{Your Name}
\date{\today}

\begin{document}
\maketitle

\section{Introduction}
Your content here...

\end{document}
```
