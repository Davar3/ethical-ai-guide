# AI Tools Overview

A guide to the main AI tools available for academic research.

## General-Purpose AI Assistants

### ChatGPT (OpenAI)

| Feature | Details |
|---------|---------|
| **Best For** | Writing, brainstorming, code |
| **Cost** | Free (GPT-3.5) / $20/mo (GPT-4) |
| **Special** | Deep Research Mode |
| **URL** | [chat.openai.com](https://chat.openai.com) |

!!! tip "Deep Research Mode"
    Searches the web and synthesizes comprehensive reports. Great for literature overviews!

---

| Feature | Details |

### DeepSeek

| Feature | Details |

## Which Tool for Which Task?

| Task | Recommended Tool |
|------|------------------|
| Brainstorming titles | ChatGPT, Claude |
| Literature overview | Gemini Deep Research |
| Analyzing your papers | NotebookLM |
| Free alternative | DeepSeek |
| Cited web search | Perplexity |
| Reference management | z.ai + Zotero |

!!! warning "Remember"
    No single tool is perfect. Experiment to find what works best for your workflow!
